# AndroidTVKeycodeController
安卓TV设备手机遥控器和接收器
### 功能
支持遥控器的所有按键
支持遥控器的鼠标键
支持记忆IP功能
支持长按关机键关机短按重启功能
### 开发计划
当该项目star超过10个的时候, 优化鼠标键功能, 让鼠标键可以根据连续的"上下左右"按键, 加速鼠标指针移动
### 使用方法
* 关机键: 短按重启, 长按关机
* 鼠标键: 按一次显示鼠标指针, 按两次关闭鼠标指针
* 其他按键: 和普通遥控器一样
### 缺少功能
长按Home按键调出最近运行的程序功能
